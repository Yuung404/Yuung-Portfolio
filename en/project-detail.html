DESCRIPTION

The MADZI Project is a network infrastructure lab carried out as part of the BTS SIO programme. The goal was to design and configure a complete enterprise network with VLAN segmentation, dynamic OSPF routing between several Cisco routers, and the deployment of DHCP and DNS network services.

OBJECTIVES

Configure a multi-site network infrastructure on Cisco equipment: create and assign VLANs, set up inter-VLAN routing via sub-interfaces on a router, enable the OSPF dynamic routing protocol between three routers (R1, R2, R3), and deploy DHCP pools with automatic IP address assignment to client workstations.

IMPLEMENTATION

The project was carried out in several stages:

1. VLAN Configuration — Creating and assigning VLANs on switches, configuring ports in access and trunk mode.

2. Inter-VLAN Routing — Configuring sub-interfaces on the router with 802.1Q encapsulation, assigning gateway addresses for each VLAN.

3. OSPF — Enabling OSPF on all three routers, declaring networks in area 0. Resolved a clock conflict on R1 using the clock rate command.

4. DHCP / DNS — Configuring DHCP pools on the router, verifying automatic IP address assignment on client workstations.

RESULTS

The infrastructure was successfully deployed. Ping tests between R1, R2 and R3 are conclusive after resolving the clock conflict. Client machines correctly obtain their IP addresses via DHCP. This project strengthened skills in Cisco network administration, OSPF dynamic routing and VLAN segmentation.
