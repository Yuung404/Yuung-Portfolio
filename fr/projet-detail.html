DESCRIPTION

Le Projet MADZI est un TP d'infrastructure réseau réalisé dans le cadre du BTS SIO. L'objectif était de concevoir et configurer un réseau d'entreprise complet avec segmentation par VLANs, routage dynamique OSPF entre plusieurs routeurs Cisco, et mise en place des services réseau DHCP et DNS.

OBJECTIFS

Configurer une infrastructure réseau multi-sites sur équipements Cisco : créer et attribuer des VLANs, mettre en place le routage inter-VLAN via des sous-interfaces sur un routeur, activer le protocole de routage dynamique OSPF entre trois routeurs (R1, R2, R3), et déployer les pools DHCP avec attribution automatique des adresses IP aux postes clients.

RÉALISATION

Le projet a été mené en plusieurs étapes :

1. Configuration des VLANs — Création et attribution des VLANs sur les switches, configuration des ports en mode access et trunk.

2. Routage inter-VLAN — Configuration des sous-interfaces sur le routeur avec encapsulation 802.1Q, attribution des adresses de passerelle pour chaque VLAN.

3. OSPF — Activation du protocole OSPF sur les trois routeurs, déclaration des réseaux dans la zone 0. Résolution d'un conflit d'horloge sur R1 via la commande clock rate.

4. DHCP / DNS — Configuration des pools DHCP sur le routeur, vérification de l'attribution automatique des adresses IP sur les postes clients.

RÉSULTATS

L'infrastructure a été déployée avec succès. Les tests de ping entre R1, R2 et R3 sont concluants après résolution du conflit d'horloge. Les machines clientes obtiennent correctement leurs adresses IP via DHCP. Ce projet a permis de consolider les compétences en administration réseau Cisco, en routage dynamique OSPF et en segmentation VLAN.
